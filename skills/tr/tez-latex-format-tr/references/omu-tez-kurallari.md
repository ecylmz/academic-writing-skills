# OMU Thesis Formatting Rule Summary

This is a working summary, not a substitute for the user's local OMU template or current institute guide. Treat every finding as source-qualified.

## Source Categories

| Category | Use when | How to report |
|---|---|---|
| Local template evidence | A `.cls`, `.sty`, main `.tex`, sample thesis, or build file defines the behavior | Cite the file and command or setting |
| OMU guide summary | Only this summary is available | Mark as `summary-based`; avoid definitive institutional wording |
| Uncertain rule | The local template and summary differ, or neither provides enough evidence | Mark as `requires current OMU/template check` |

## Working Rules

- A4 paper.
- 4 cm left margin and 2.5 cm other margins.
- Bottom-centered page numbers.
- Roman numerals before main text.
- Times New Roman 12 pt body text.
- 14 pt main headings.
- 1.5 body line spacing.
- Single spacing in summaries, lists, references, and CV.
- Chapter-numbered figures, tables, and equations.
- Table captions above tables.
- Figure captions below figures.
- References alphabetized and consistently styled.

Preserve local template behavior when present. If the template and this summary differ, report the conflict instead of declaring a violation.
