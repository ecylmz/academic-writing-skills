# Paper Workflow Example

## 1. Introduction revision

```text
Use makale-yazimi-en to revise this Introduction. Make the task-gap-contribution logic clearer and preserve all citations.
```

## 2. Claim-evidence map

```text
Run claim-evidence-audit on the Abstract and Introduction. Flag unsupported and overclaimed statements.
```

## 3. Reviewer-risk audit

```text
Use makale-denetim-en as a skeptical reviewer. Focus on methodology, evaluation completeness, and defect prediction protocol risks.
```

## 4. Style review

```text
Use academic-style-review-en to remove vague claims, filler, and AI-like phrasing without adding facts or citations.
```

## 5. Citation audit

```text
Run citation-integrity-audit on manuscript.tex and references.bib. Report missing references, orphan references, DOI gaps, and source-support risks.
```
