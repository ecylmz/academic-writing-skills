# Paper Workflow Example

## 1. Introduction revision

```text
Use paper-writing-en to revise this Introduction. Make the task-gap-contribution logic clearer and preserve all citations.
```

## 2. Claim-evidence map

```text
Run research-integrity-audit in claim-evidence mode on the Abstract and Introduction. Flag unsupported and overclaimed statements.
```

## 3. Reviewer-risk audit

```text
Use paper-review-en as a skeptical reviewer. Focus on methodology, evaluation completeness, and defect prediction protocol risks.
```

## 4. Style review

```text
Use paper-writing-en in style-review mode to remove vague claims, filler, and AI-like phrasing without adding facts or citations.
```

## 5. Citation audit

```text
Run research-integrity-audit in citation-integrity mode on manuscript.tex and references.bib. Report missing references, orphan references, DOI gaps, and source-support risks.
```
