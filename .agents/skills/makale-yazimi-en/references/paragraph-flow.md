# Paragraph Flow Guide

## Flow test

For each paragraph, check:

1. What is the paragraph's main message?
2. Is the message visible in the first sentence?
3. Does each sentence add cause, contrast, evidence, example, or consequence?
4. Does the final sentence either complete the paragraph or connect to the next one?
5. Are terms consistent?

## Reverse outline

Create a table:

| Paragraph | Role | Topic sentence | Evidence | Problem |
|---|---|---|---|---|

## Repair strategies

- Split paragraphs with two messages.
- Merge paragraphs that repeat the same point.
- Move background sentences before technical claims.
- Replace vague transitions with logical relations.
- Remove summary closers that do not add information.
