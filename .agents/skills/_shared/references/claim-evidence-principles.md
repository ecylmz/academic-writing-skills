# Claim-Evidence Principles

Akademik metinde güvenilirlik, iddianın gücünün kanıt türüyle uyumlu olmasına bağlıdır.

## Claim türleri

| Tür | Örnek | Gerekli destek |
|---|---|---|
| Olgusal iddia | `X veri kümesi 16 projeden oluşur.` | Veri kümesi açıklaması, tablo, kaynak veya yöntem bölümü |
| Nicel iddia | `F1 skoru 0.762'dir.` | Sonuç tablosu, deney çıktısı, istatistiksel rapor |
| Karşılaştırmalı iddia | `Yöntem A, B'den daha iyi sonuç vermiştir.` | Aynı protokol altında karşılaştırma tablosu |
| Nedensel iddia | `Dengeleme yöntemi performansı artırmıştır.` | Uygun deney tasarımı, ablation, kontrol değişkenleri |
| Yorumlayıcı iddia | `Bu sonuç transfer öğrenmenin sınırlı kaldığını düşündürmektedir.` | Bulgular + alternatif açıklamalar + sınırlılık |
| Katkı iddiası | `Bu çalışma Go için ilk veri kümesini sunar.` | Literatür taraması ve kapsam sınırı |

## Kalibrasyon

- `gösterir / demonstrates`: güçlü ve doğrudan kanıt varsa kullan.
- `işaret eder / suggests`: sonuç destekli ama sınırlı genelleme varsa kullan.
- `uyumludur / is consistent with`: bulgu literatürle örtüşüyorsa kullan.
- `kanıtlar / proves`: teorik ispat yoksa kullanma.
- `en iyi / the best`: tüm güçlü alternatifler aynı koşulda karşılaştırılmadıysa kullanma.

## Eksik destek durumunda yapılacaklar

1. İddia korunacaksa destek eklenmelidir.
2. Destek yoksa iddia zayıflatılmalıdır.
3. Zayıflatılamıyorsa iddia silinmelidir.
4. Metinde kalması gerekiyorsa sınırlılık olarak sunulmalıdır.
