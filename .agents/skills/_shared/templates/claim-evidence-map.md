# Claim-Evidence Map Template

| # | Section | Claim | Claim Type | Evidence | Evidence Location | Status | Required Action |
|---|---|---|---|---|---|---|---|
| 1 |  |  | factual / quantitative / comparative / causal / interpretive / contribution |  | table / figure / citation / experiment / method | supported / weak / missing / overclaimed | keep / weaken / add evidence / remove |

## Verdict labels

- `supported`: claim and evidence match.
- `weak`: evidence exists but the wording is too strong.
- `missing`: no evidence is provided.
- `overclaimed`: evidence supports a narrower claim.
- `misaligned`: cited evidence supports a different point.
